# Cançoner Català

Aplicació web per a la consulta i gestió de la base de dades bibliogràfica del Cançoner Català.

## Descripció

El Cançoner Català és un projecte de bibliografia que recull informació sobre cançoners, folklore, danses i altres documents relacionats amb la cultura catalana. Aquesta aplicació web proporciona una interfície moderna per accedir a aquesta informació històrica.

## Contingut de la base de dades

- **Total de registres**: 467
- **Tipus de documents**:
  - Cançoner
  - Folklore
  - Dansa
  - Altres

## Tecnologies

- **Backend**: Flask (Python)
- **Base de dades**: SQLite
- **Servidor**: Raspberry Pi 2 (Raspbian 11)
- **Desplegament**: Podman
